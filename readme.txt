A starter theme called oneBlog.

==== Description ====

oneBlog is a new wordpress theme that allow to create dynamic blogs. This is very well designed and developed theme with responsive desing.

==== Installation ====

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now or type oneBlog in search box and install.
3. Click Activate to use your new theme right away.
